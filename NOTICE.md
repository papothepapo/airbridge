# Notice

Airbridge was developed with reference to publicly available AirPods reverse-engineering work and on-device validation against a rooted Android 9 target.

Public projects and research that were useful during protocol bring-up:

- LibrePods
- Capod
- public community notes around Apple BLE manufacturer packets and the AirPods AAP/AACP control channel

This repository does not include third-party source trees or copied build artifacts. If you import external code later, preserve that code's original license and attribution.
